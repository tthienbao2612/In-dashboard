Kết nối với Google Sheets

Dán URL Web App từ Google Apps Script vào đây để dashboard tự động lấy dữ liệu realtime mỗi 30 giây.

  1. Mở file Sheets → Extensions → Apps Script
  2. Tạo file Code.gs mới, dán nội dung từ file Code.gs đi kèm
  3. Nhấn Deploy → New deployment → Web App
  4. Đặt Execute as: Me · Who has access: Anyone
  5. Nhấn Deploy → Copy URL → Dán vào ô bên dưới
URL được lưu trong trình duyệt, không cần nhập lại lần sau.
